🍽️ Maddy's Restaurant Website
📌 Project Title

Maddy's Restaurant – Online Food Ordering Website

📖 Description

Maddy's Restaurant is a simple and interactive food ordering website built using HTML, CSS, and JavaScript. It allows users to browse food items, filter between veg and non-veg dishes, add items to a cart, and complete orders using UPI payment or cash option.

This project is beginner-friendly and demonstrates core front-end web development concepts.

🚀 Features
🏠 Attractive homepage with restaurant title
🥗 Filter menu (Veg / Non-Veg / All)
🛒 Add items to cart with quantity selection
📄 Dynamic bill generation
💳 UPI QR code payment system
💵 Cash payment option
🔄 Auto refresh after order completion
📱 Responsive design
🛠️ Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout
JavaScript – Functionality and interactivity

💡 How It Works
Users select food items from the menu
Enter quantity → item gets added to cart
Click View Bill to see total
Choose:
Scan QR & Pay (UPI)
Cash Payment
After payment → order confirmed → page reloads
